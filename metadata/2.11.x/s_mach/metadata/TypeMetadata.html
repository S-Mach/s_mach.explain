Related Docs: object TypeMetadata | package metadata

sealed trait TypeMetadata[+A] extends MetaTree[A, TypeMetadata]

Base trait for a node in a tree of type metadata. A tree of type metadata allows associating metadata values with specific fields, array types or values contained in a the serialized output of a type. Commonly used to by validation code to communicate the set of possible checks for a type to callers.
